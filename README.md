# my-11th-project

This example demonstrates a simple function that performs an arithmetic operation and returns a value. It highlights fundamental JavaScript concepts such as function definition, parameters, return statements, variable assignment, and console output.

 Code
function add(a, b) {
    return a + b;
}

let result = add(5, 3);
console.log("The result is: " + result);

 Overview

The add function is defined with two parameters: a and b.

The function returns the sum of the two input values.

The returned value is stored in the variable result.

The final result is printed to the console.
